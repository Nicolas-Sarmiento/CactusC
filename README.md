# 🌵 CactusC — A Tiny C-Like Language  
### Compiled by 🐟 ctuna

**Cactus** is a small experimental programming educational language inspired by C .  
**ctuna** is its lightweight compiler, written entirely in C — built to understand compilation from the inside out.

This project is a learning experience and a playground for exploring the stages of compilation, from raw source code to tokens, syntax trees, and beyond.

---

## 🚀 Goals

- ✅ Minimal CLI compiler
- 🔄 Lexical analysis via Mealy/Moore machines
- 🔄 Handwritten parser for a tiny C-like grammar
- 🔄 AST generation
- 🔲 Semantic analysis
- 🔲 Optional: output to C or bytecode
